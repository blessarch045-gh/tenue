## 2026年アルゴリズム 重要事実

### Instagram
- **視聴時間 > DMシェア > 保存 > いいね** の順で評価
- キーワードSEOはハッシュタグより **30%多いリーチ**
- リールのリーチ率 **30.81%**（カルーセルの1.36倍）
- コラボ投稿は通常の **3.8倍のリーチ**
- 新規アカウントの **最初の10投稿は2倍のリーチブースト**

### X
- **リプライ往復（会話成立）= いいねの150倍の重み**
- 動画投稿はテキストの **2〜3倍のインプレッション**
- 外部リンクを本文に貼ると **最大40%のペナルティ**
- ハッシュタグ **4つ以上でスパム判定リスク**

## Instagram 運用方針

### コンテンツミックス（週5〜7投稿）
| 曜日 | フォーマット | コンテンツ |
|------|------------|----------|
| 月 | リール（15〜30秒） | 1着で3コーデ着回し |
| 火 | カルーセル | 保存されやすい実用系（「春コーデ5選」等） |
| 水 | リール（15〜30秒） | GRWM / Before→After |
| 木 | - | エンゲージメント活動のみ（コメント回り） |
| 金 | リール（15〜30秒） | 季節・トレンド系 |
| 土 | カルーセル | 世界観重視のビジュアル |
| 日 | ストーリーズ集中 | 週振り返り＋来週予告 |

### 毎日のルーティン（30分）
| 時間 | アクション |
|------|----------|
| 朝5分 | ストーリーズ（投票・質問スタンプで双方向性） |
| 昼5分 | 5-3-1ルール（いいね5件・深いコメント3件・フォロー1件） |
| 夜20分 | メイン投稿（リール or カルーセル） |

### キャプション SEO最適化フォーマット
```
[最初の125文字: 強いフック＋ターゲットキーワード]
例: 「通勤コーデ、毎朝迷いませんか？この1着で月〜金着回せます」

[本文: 共感・ストーリー]

[CTA: 保存・シェアを促す]
例: 「保存して明日のコーデの参考にしてね」

[ハッシュタグ: 3〜5個]
#コーデ記録 #今日の服 #着回しコーデ #tenue記録
```

### やってはいけないこと
- ハッシュタグ6個以上（スパム判定）
- 画像なしの投稿（エンゲージメント最低）

## X 運用方針

### コンテンツの4本柱
| カテゴリ | 比率 | 例 |
|---------|------|---|
| 共感・日常 | 40% | 「朝、何着るかで1日の気分が変わる」 |
| コーデTips | 30% | 着回し術、アイテム合わせのコツ |
| 世界観 | 20% | 「装いの記憶」テーマの物語ツイート |
| 参加型 | 10% | 「今日のコーデのポイントは？」二択質問 |

### 毎日のルーティン（20分）
| 時間 | アクション |
|------|----------|
| 朝5分 | 共感ツイート1本（世界観ツイート） |
| 昼10分 | ファッション系アカウントに質の高いリプライ10〜15件 |
| 夜5分 | 引用RT 1〜2件（世界観に沿った一言添え） |

### やってはいけないこと
1. ハッシュタグ4つ以上（スパム判定リスク）
2. 外部リンクを本文に貼る（代わりに「プロフのリンクから」）
3. 大量フォロー（スパム認定・FF比率悪化）
4. 開発者目線の投稿（「アプリ開発中」「バグ修正完了」等はNG）
5. Instagramの投稿をそのまま転載（プラットフォーム最適化が必要）
