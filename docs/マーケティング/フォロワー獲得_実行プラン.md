# tenue フォロワー獲得 実行プラン

作成日: 2026-03-15
前提: 広告予算ゼロ / 既存知名度なし / ファッション×若い女性ターゲット / 開発者目線は出さない

---

## 調査で判明した最重要ファクト

### Instagram（2026年アルゴリズム）
- **視聴時間 > DMシェア > いいね比率** の順で評価される
- 「いいね」は最も弱いシグナル。**保存・DMシェア・会話**が重要
- **キーワードSEO がハッシュタグより30%多いリーチ**を獲得
- リールのリーチ率 **30.81%**（カルーセルの1.36倍）
- コラボ投稿は通常の **3.8倍のリーチ**
- **新規アカウントの最初の10投稿は2倍のリーチブースト**がかかる

### X（2026年アルゴリズム）
- 2026年1月にGrok AIベースに完全移行
- **リプライの往復（会話成立）= いいねの150倍の重み**
- 動画投稿はテキストの **2〜3倍のインプレッション**
- **外部リンクは最大40%のペナルティ**（本文に貼らない）
- ハッシュタグ5つ以上で**スパム判定**リスク

### 参考事例
- **古着女子**: 広告費ゼロ・5ヶ月で10万フォロワー。UGCリポスト戦略
- **Daniel Wellington**: マイクロインフルエンサーへのギフティングのみで急成長
- **Lemon8**: エンゲージメント率約6%。競合少なく初期獲得に有効

## フェーズ2: 100→500フォロワー（2〜3ヶ月目）

### 追加施策

#### 1. コラボ投稿（Instagram）
- 同規模（フォロワー100〜1000人）のコーデ系アカウントを探す
- 「お互いのフォロワーに紹介し合う」コラボリールを月2〜4回
- コラボ投稿はリーチ3.8倍

#### 2. UGC戦略の開始
- **#tenue記録** ハッシュタグを育てる
- βテスト参加者にハッシュタグ付きコーデ投稿を依頼
- 良質な投稿を公式でリポスト（古着女子モデル）
- 「リポストされること＝ステータス」の構造を作る

#### 3. スレッド投稿（X）
- 週1回、ファッションTips系のスレッドを投稿
- 例: 「1週間着回しコーデの考え方」5ツイートスレッド
- スレッドは単体ツイートの3倍のエンゲージメント

#### 4. 動画のクロスプラットフォーム展開
- Instagramリール用素材を60秒以内に再編集してXにも投稿
- 1素材で2プラットフォームに展開（制作コスト削減）

## フェーズ3: 500→1000フォロワー（4〜6ヶ月目）

- ストーリーズの投票機能で選出
- 選ばれた人を公式で紹介（UGC＋コミュニティ感）

## リールの鉄板フォーマット

### 最初の3秒（フック）のテンプレ

| タイプ | 例文 |
|--------|------|
| 実用性 | 「この1着で1週間着回せます」 |
| 共感 | 「朝クローゼットの前で固まったこと、ある？」 |
| 価格 | 「全身5,000円以下の通勤コーデ」 |
| 変身 | 「部屋着→おでかけコーデ、10秒チェンジ」 |

### 構成（15〜30秒）
1. テキスト付きフック（3秒）
2. コーデ紹介・切り替え（10〜20秒）
3. CTA「保存してね」「友達にもシェア」（3秒）

### 注意点
- 80%以上が無音視聴 → **テキスト字幕は必須**
- 90秒以内が推奨（非フォロワーへのレコメンド対象になる）
- 完視聴率60%以上を目標に

## Xで絶対やってはいけないこと

1. **ハッシュタグ4つ以上**（スパム判定リスク）
2. **外部リンクを本文に貼る**（インプレッション最大40%減）
   → 代わりに「プロフィールのリンクから」と誘導
3. **大量フォロー**（スパム認定＋FF比率悪化）
4. **開発者目線の投稿**（「アプリ開発中」「バグ修正完了」等はNG）
5. **Instagramの投稿をそのまま転載**（各プラットフォーム最適化が必要）
