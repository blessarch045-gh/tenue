# X リプライ戦略書 — 「コーデ記録キャラクター」でフォロワーを増やす

> 作成日: 2026/3/16
> 戦略概要: バズり投稿へのリプライ経由でインプレッションを獲得し、「コーデを記録する人」キャラクターを確立してフォロワーを増やす

---

## なぜリプライ戦略が機能するか（メカニズム）

```
バズ投稿（10万imp）
    ↓
質の高いリプライ（1〜5万imp流入）
    ↓
「この人詳しそう」→ プロフクリック
    ↓
「コーデ記録アプリ作ってる人か」→ フォロー
```

AI系X投稿（「Claudeで自動化した」）が同じ構造で機能している。
ファッション版では **「コーデを記録し続けているから見えてくる視点を持っている人」** がそのポジション。

---

## 戦略の核：「コーデ記録している人」キャラクターの確立

| AI系バズ投稿 | tenue版 |
|---|---|
| 「Claudeで自動化した」 | 「毎日コーデを記録している」 |
| 具体的なプロンプト公開 | 具体的な記録データ・気づき公開 |
| 「私もできそう」 | 「面白い視点」「自分もやりたい」 |
| フォローしたら役立つ | フォローしたら独自の視点が得られる |

**一貫して出すアイデンティティ**: 「毎日コーデを記録しているから、普通の人が気づかないことに気づいている人」

---

## リプライテンプレート 3パターン

### パターンA：「体験データ型」（最強）
バズ投稿に対して**自分の記録データ**を具体的に添える

```
ターゲット例：「春はベージュコーデが最強説」

リプライ例：
去年の春コーデ全部記録してたら、
自分が選んでた色の8割がベージュ系でした笑

意識してないのに手が勝手に選んでる色って、
ほんとうに「自分の色」なんだなと思った。
```
✅ 「コーデを記録している人」キャラクター自然に確立
✅ 「8割」という数字がフック
✅ 共感した人がプロフを見に来る

---

### パターンB：「視点追加型」
バズ投稿への賛同 ＋ **一段深い観察**を添える

```
ターゲット例：「このワンピース最高すぎる」

リプライ例：
このシルエット、年中使えるんですよね。

私が記録を見返すと夏も秋も冬も登場してて、
結局1着で100日以上着てた。
コスパ最強アイテムの条件って
「季節またいで着られるか」だと気づいた。
```
✅ 「記録して検証している人」の信頼性
✅ 「100日以上」の具体数字がフック

---

### パターンC：「問いかけ型」
会話が生まれやすく、**リプライ往復でアルゴリズム評価UP**（X最重要指標）

```
ターゲット例：「衣替えが一番ワクワクする季節」

リプライ例：
わかります。箱を開けると去年の自分に会う感じ。

私は毎日コーデ記録してるので
去年の今頃何着てたか正確にわかるんですが、
今年と全然違うアイテム選んでてびっくりします。

去年の春、何色が多かったですか？
```
✅ 最後の問いかけで相手が返しやすい
✅ 返信が来るとインプレッション倍増（Xアルゴリズム最重要指標）

## バズり投稿の見つけ方・タイミング

**鉄則：投稿後30〜60分以内にリプライ**

```
発見方法：
1. 「おすすめ」タブを1日3回チェック（朝・昼・夜）
2. フォローしているファッション系アカウントのTL
3. 検索「#春コーデ」「#今日のコーデ」→「最新」タブで急上昇を探す

見極め基準（返信する価値あり）：
- いいね 100〜1,000件（伸び始め）
- リプライが少ない（競合が少ない）
- 共感・Tips・発見系（ブランド広告は除く）
```

### 優先ターゲットアカウント（リプライ先として有効）
| カテゴリ | 例 |
|---|---|
| ファッション系インフルエンサー（1万〜10万フォロワー） | コーデ系・着回し系 |
| ファッション好きの一般ユーザー（1,000〜5,000フォロワー） | バズり始めが狙い目 |
| ファッションブランド公式アカウント | 共感リプライが刺さりやすい |

## やってはいけないこと

| NG | 理由 |
|---|---|
| 「すごい！参考になります！」だけ | 価値ゼロ・スパム同然 |
| 「アプリ作ってます」「tenue見てください」 | 宣伝臭で即スルー |
| 同じ文面を複数投稿に使い回す | スパム判定・信頼失墜 |
| ハッシュタグ4つ以上をリプライに入れる | スパム判定リスク |
