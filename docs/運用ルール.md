# tenue 運用ルール

Claude（AI）が作業する際に必ず守るルール集。CLAUDE.md から詳細を分離。

---

## ブラウザ・自動化ルール

- **詰まったら即ストップ**: ブラウザ操作・自動化作業がスムーズに進まない場合は粘らず中断し、ユーザーに状況を報告して指示を仰ぐ
- **金銭操作は禁止**: 購入・課金・支払い・サブスクリプション登録が発生したら即停止・報告（Apple Developer Program登録、広告費用、有料プランアップグレード等）
- **返信・投稿ボタンは自分で押さない**: X の「返信」ボタン、Instagram の「シェア」ボタン等、公開投稿を確定するボタンは必ずユーザーが押す

---

## SNS 運用ルール

### 共通
- 投稿・返信の最終確定はユーザーが行う（AIは入力・下書きまで）
- **Instagram投稿確認を繰り返さない**: ユーザーが「投稿済み」と伝えた投稿について、再度「投稿できていない」と確認しない。ユーザーの申告を信頼する

### X（ツイッター）リプライ
- **公式アカウントのみ**: 個人アカウントは絶対に対象外。ブランド・アパレル・ファッションメディアの公式アカウントに限定
- **女性向けコンテンツのみ**: 男性向けファッション・子供向けコンテンツはスキップ（tenuеのターゲットは女性）
- **返信ボタンはユーザーが押す**: 入力まで行い、送信はユーザーに委ねる
- 詳細手順・アカウントリスト → `docs/スキル/スキル_Xリプライ.md`

## Cowork（AI）活用方針

### AIが自動化する作業
- コンテンツ企画・キャプション生成・画像プロンプト作成
- ツイート案作成・リプライ下書き
- 分析レポート・ファイル整理

### ユーザーが手動で行う作業
- Instagram投稿（Googleドライブから画像取得→投稿）
- X投稿・返信ボタンのクリック
- YouTube動画アップロード
- Apple Developer Program登録・その他課金操作
