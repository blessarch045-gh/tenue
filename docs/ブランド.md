# ブランド設計
> このファイルの用途: UI文言・SNSコピー・ストア掲載文を作成するときのトーン・カラー・表現ルール。

## 基本情報
- 正式名称: tenue（全小文字）
- 読み方: トゥニュ（仏語）／テニュ（日本語読み）
- 意味: フランス語で「装い・コーディネート」
- ミッション: 服と、人と、その日を残す。
- ターゲット: 10〜30代女性

## トーン・声のルール
- やさしく、押しつけない。「〜してみない？」「〜してみよう。」
- 日常の小さな発見を大切にする語り口
- フランス語の由来を活かしつつ、堅くならない

## 避けるべき表現
- 「最強」「神」「バズる」などの煽り表現
- 「ダウンロードしてね！」「フォローしてね！」など押しつけがましいCTA
- 開発者視点の表現（「バグ修正」「実装完了」「アプリ開発中」）
- 他アプリの批判や比較（「〇〇よりいい」）
- 過度なカタカナ英語（「マネタイズ」「グロースハック」等）
