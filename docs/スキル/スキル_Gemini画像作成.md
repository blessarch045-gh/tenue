# スキル: Gemini 投稿画像作成

> トリガー: 「（週の）画像作って」「Geminiで画像作って」「今週の画像を作って」など

---

## 概要

Instagram投稿用の画像をGemini（Google AI Studio）でバッチ生成するスキル。
投稿予定ファイルに記載されたプロンプトを使い、1回の作業で複数枚を一気に生成・保存する。

---

## 前提確認（スキル開始前に確認）

1. **対象ファイルの特定**
   - `マーケティング/Instagram/` 内の `投稿予定_[期間].md` を確認
   - 未生成の画像プロンプトが含まれているか確認
   - 既存画像の有無: `マーケティング/Instagram/画像/` 内をチェック

2. **ブラウザ状態の確認**
   - Gemini（https://gemini.google.com）またはGoogle AI Studio（https://aistudio.google.com）へのアクセスが必要
   - ログイン済みかどうかを確認

---

## 実行フロー

### ステップ1: 投稿予定ファイルを読み込む

```
対象ファイル例: マーケティング/Instagram/投稿予定_Week1_3月16日〜3月22日.md
```

- ファイルから「画像プロンプト」セクションを全て抽出
- 何枚生成が必要か（カルーセル枚数含む）を確認してユーザーに報告

例:
```
今週の画像は以下の構成です。生成を開始します。
- 3/16（月）リール表紙: 1枚
- 3/18（水）カルーセル: 5枚
- 3/20（金）静止画: 1枚
- 3/22（日）リール表紙: 1枚
合計: 8枚
```

### ステップ2: Geminiを開く

- https://gemini.google.com または https://aistudio.google.com にナビゲート
- 画像生成機能（Imagen 3）が使えるモデルを選択

### ステップ3: プロンプトを送信・画像を生成

投稿予定ファイルの「画像プロンプト」をそのまま送信。

**共通スタイル指定（プロンプトに必ず付加）:**
```
水彩タッチのイラスト風。暖色系（アイボリー・ベージュ・テラコッタ）。
少女マンガ的な繊細さ。文字なし。
```

**アスペクト比の指定:**
| 投稿フォーマット | 比率 | Gemini指定 |
|----------------|------|-----------|
| リール（動画表紙） | 9:16 縦長 | 「縦長（9:16）」 |
| カルーセル・静止画 | 1:1 正方形 | 「正方形（1:1）」 |
| ストーリーズ | 9:16 縦長 | 「縦長（9:16）」 |

### ステップ4: 画像をダウンロード・保存

**保存先:** `マーケティング/Instagram/画像/`

**ファイル命名規則:**
```
[Week番号]_[月日]_[テーマ略称]_[連番].png

例:
Week1_3月18日_色合わせ1_表紙.png
Week1_3月18日_色合わせ2_白ブラウス.png
Week2_3月23日_春の出会い.png
```

カルーセルの場合は末尾に連番をつける（_1, _2, _3...）

### ステップ5: 完了報告

生成・保存した画像ファイルの一覧をユーザーに報告:
```
✅ 画像生成完了。以下のファイルを保存しました：
- マーケティング/Instagram/画像/Week1_3月18日_色合わせ1_表紙.png
- マーケティング/Instagram/画像/Week1_3月18日_色合わせ2_白ブラウス.png
（以下続く）
```

## 画像スタイルガイド（tenue ブランド）

| 要素 | 仕様 |
|------|------|
| スタイル | 水彩タッチのイラスト風 |
| 雰囲気 | 少女マンガ的な繊細さ・温かみ |
| カラーパレット | アイボリー・ベージュ・テラコッタ・くすみピンク・ペールブルー |
| 人物 | 20〜30代の日本人女性（具体的な顔は描かず横顔・後ろ姿推奨） |
| テキスト | **なし**（Geminiはテキスト生成が不安定なため、テキストは別途追加） |
| 画質 | 高品質・Instagram映え |

---

## よくある失敗とリカバリー

| 問題 | 対処 |
|------|------|
| テキストが画像に混入する | プロンプトに「文字なし・テキストなし・no text」を追加して再生成 |
| スタイルがリアル寄りになる | 「水彩タッチ・アニメイラスト風・ソフトなタッチ」を追加 |
| カラーがブランドと合わない | 「アイボリー・ベージュ・テラコッタのみ使用」と明示 |
| 縦横比が違う | 「縦長9:16のみ生成してください」と明示 |
| Geminiが生成を拒否 | プロンプトから具体的な人物描写を減らして再試行 |

---

## 注意事項

- **ダウンロードは必ずユーザー確認後**: 生成された画像をダウンロード・保存する前に、ユーザーが確認した上で「保存して」と指示があってから保存する
- **1回あたりの生成枚数**: Geminiの制限により1プロンプトで1〜4枚程度。カルーセル5枚は複数回に分けて生成
- **再生成の判断**: 明らかにブランドイメージと違う場合は自動的に再生成。微妙な場合はユーザーに確認
