## 毎朝（5分）: 共感ツイート投稿

### 投稿ルール
- 文字数: 100〜140字（余白を残す）
- ハッシュタグ: 1〜2個まで（3個以上でスパムリスク）
- **外部リンクは本文に貼らない**（インプレッション最大40%減）
- 開発・技術・アプリ機能の話はしない

### 投稿テンプレート（ローテーション）

**月・木: 共感・日常系（40%）**
```
朝、何着るかで1日の気分が変わる。
服はただの布じゃなくて、今日の自分を決めるもの。

#コーデ記録
```

**火・金: コーデTips系（30%）**
```
着回しのコツは「主役を1つ決めること」。
バッグを主役にしたら、服は引き算でいい。

#ファッション好き
```

**水・土: 世界観・ブランド系（20%）**
```
「あの日、何着てたっけ」
思い出せなくて、少し寂しくなる。
装いは記憶になる。

── tenue
```

**日: 参加型（10%）**
```
今日のコーデのポイント、どっち派？

A. 色で遊ぶ
B. シルエットで決める

#ファッション好き
```

## 昼（10分）: リプライ回り

### 検索キーワード（毎日ローテーション）
| 曜日 | 検索ワード |
|------|----------|
| 月 | #今日のコーデ |
| 火 | #コーデ記録 |
| 水 | #春コーデ（季節に合わせて変更） |
| 木 | #ファッション好き |
| 金 | #ファッション好きな人と繋がりたい |
| 土 | #おしゃれさんと繋がりたい |
| 日 | #着回しコーデ |

### リプライの書き方（10〜15件/日）

**良いリプライの例:**
```
素敵なコーデですね。テラコッタのニット、今季本当に使いやすそうで。
```
```
このシルエット好きです。インナーの色の選び方が絶妙。
```
```
雨の日にこの明るい色、気分上がりますよね。
```

**NGリプライ:**
- 「フォローお願いします」「相互フォロー」→ スパム扱い
- 1〜2文字の短いリプライ（「いいね！」等）→ 重み低い
- アプリ宣伝・外部リンク貼り付け→ 絶対NG

### リプライ品質チェック
- [ ] コーデの具体的な要素（色・アイテム・コーデの工夫）に触れているか
- [ ] 30字以上あるか
- [ ] 宣伝・誘導になっていないか

## 夜（5分）: 引用RT

### 検索して素敵な投稿を1〜2件引用RT

**引用RTの書き方:**
```
この着こなし方、参考になります。
ネイビーとテラコッタの組み合わせ、今季やってみたい。
```
```
春の訪れを服で感じるの、すごく好きです。
```

**引用RT時のルール:**
- コーデへの共感・感想を一言添える
- 宣伝にならないこと
- ハッシュタグなしでOK

## やってはいけないこと

1. **ハッシュタグ3個以上**（2個以内が安全）
2. **本文に外部リンクを貼る**（プロフィールリンクに集約）
3. **大量フォロー**（1日10人以上はスパム認定リスク）
4. **開発者目線の投稿**（「バグ修正」「機能追加中」等）
5. **Instagramの投稿をそのまま転載**（最適化が必要）
6. **毎回同じ文型の投稿**（バリエーションをつける）
